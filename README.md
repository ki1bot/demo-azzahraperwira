# Project Yayasan V5

Project Yayasan V5 adalah aplikasi website profil yayasan berbasis CodeIgniter 4. Aplikasi ini dibuat untuk menampilkan informasi yayasan, profil lembaga, tenaga pengajar, unit pendidikan, informasi kegiatan, serta menyediakan halaman admin untuk mengelola konten website.

## Fitur Utama

- Halaman Beranda
- Halaman Profile Yayasan
- Halaman Tenaga Pengajar
- Halaman Unit KB/TK
- Halaman Unit TPQ
- Halaman Unit Daycare
- Halaman Unit Lansia
- Halaman Informasi
- Detail informasi atau berita
- Login admin
- Dashboard admin
- Kelola konten halaman
- Tambah, ubah, dan hapus konten tertentu
- Ubah password admin
- Logout admin

## Teknologi yang Digunakan

- PHP
- CodeIgniter 4
- MySQL
- HTML
- CSS
- JavaScript
- Composer

## Kebutuhan Sistem

Pastikan perangkat sudah memiliki:

- PHP 8.1 atau lebih baru
- Composer
- MySQL atau MariaDB
- Web server seperti Apache, Laragon, XAMPP, atau server bawaan CodeIgniter
- Ekstensi PHP:
  - curl
  - intl
  - json
  - mbstring
  - mysqlnd
